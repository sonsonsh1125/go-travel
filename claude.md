# Go-Travel 프로젝트 명세서

## 프로젝트 개요

YouTube에서 특정 여행지 관련 영상을 검색하고, 각 영상의 설명(description)에서 언급된 장소들을 AI로 자동 추출하여 웹 화면에 표시하는 시스템

> **중요 변경사항**: Vercel 서버리스 환경 제약으로 인해 youtube-transcript 대신 영상 설명을 사용합니다.

## 핵심 기능

### 1. 웹 인터페이스
- Next.js 15 기반 반응형 웹 애플리케이션
- 사용자가 여행지 입력 (예: "교토", "후쿠오카", "오사카")
- 실시간 진행 상황 표시
- 검색 결과를 카드 형식으로 표시

### 2. YouTube 영상 검색
- 입력된 여행지를 기반으로 YouTube 검색
  - 검색 쿼리: "{여행지} 여행", "{여행지} 브이로그", "{여행지} 관광"
- **필터링 조건**:
  - 조회수 상위 10개 영상 (Vercel 타임아웃 최적화)
  - 타이틀에 검색어가 포함된 영상만 선택
- 수집 정보: 영상 ID, 제목, 채널명, 조회수, URL, 설명

### 3. 영상 설명(Description) 수집
- YouTube Data API v3 사용
- 각 영상의 설명 텍스트 추출
- 설명이 50자 미만인 영상은 자동 스킵
- **Vercel 최적화**: youtube-transcript 대신 description 사용

### 4. AI 장소 추출 (핵심 기능)
- **OpenAI GPT-4o-mini** 사용
- 영상 설명에서 **구체적인 상호명이 있는 장소**만 추출
- **엄격한 필터링 규칙**:
  - ✅ 구체적인 상호명 (예: "이치란 라멘", "니시키 시장")
  - ❌ 일반 지역명 제외 (예: "교토", "오사카")
  - ❌ 호텔/숙박시설 제외
  - ❌ 검색 지역 외부 장소 제외
  - ❌ 애매한 표현 제외 (예: "그 카페", "유명한 곳")
- **처리 속도**: AI 요청 간 300ms 딜레이 (Vercel 최적화)

### 5. 결과 표시 + 구글 지도 연동
- 추출된 장소를 웹 화면에 카드 형식으로 표시
- 각 장소마다 다음 정보 제공:
  - 장소명
  - 카테고리 (맛집/관광지/사찰/카페/쇼핑/액티비티/기타)
  - 설명 (자막에서 추출)
  - 언급 횟수
  - 언급된 영상 제목
  - **📍 지도 버튼** → 클릭 시 구글 지도에서 검색

## 기술 스택

## 실행 흐름

```
1. 사용자가 웹에서 여행지 입력 (예: "교토")
   ↓
2. YouTube 검색 (타이틀 필터링 + 조회수 상위 10개)
   ↓
3. 영상 설명 수집 (YouTube Data API)
   ↓
4. AI 장소 추출 (OpenAI GPT-4o-mini)
   - 구체적인 상호명만 추출
   - 호텔/일반 지역명/외부 지역 제외
   - 300ms 딜레이로 순차 처리
   ↓
5. 중복 제거 및 언급 횟수 계산
   ↓
6. 웹 화면에 결과 표시
   - 카테고리별 분류
   - 구글 지도 링크 제공
```

## AI 프롬프트 전략

### 시스템 메시지
```
당신은 여행 영상 자막에서 구체적인 장소를 추출하는 전문가입니다.
실제 방문 가능한 장소만 추출하고, JSON 형식으로 응답합니다.
```

### 사용자 프롬프트 (핵심 규칙)
```
다음은 "{destination}" 여행 영상의 설명입니다.
이 설명에서 언급된 "{destination}" 지역 내의 구체적인 장소만 추출해주세요.

다음 규칙을 반드시 지켜주세요:
1. ✅ 구체적인 상호명이 있는 장소만 추출
2. ❌ 일반적인 지역명은 제외
3. ❌ "{destination}" 외부 지역의 장소는 제외
4. ❌ 호텔/숙박 시설은 제외
5. ❌ 애매한 표현 제외
6. ✅ 실제 방문 가능한 구체적인 상호가 있는 장소만 포함
7. ✅ "{destination}" 시내 또는 근교(30분 이내)의 장소만 포함

참고: 영상 설명의 처음 8000자만 분석합니다.
```

## 카테고리 분류

- **맛집**: 레스토랑, 라멘집, 식당
- **카페**: 커피숍, 디저트 카페
- **관광지**: 유명 관광 명소, 전망대, 공원
- **사찰**: 신사, 절, 종교 시설
- **시장**: 전통 시장, 재래시장
- **쇼핑**: 쇼핑몰, 백화점, 상점가
- **액티비티**: 체험 시설, 오락실
- **기타**: 역, 공원, 기타 장소

## 비용 예상

### YouTube API
- 무료: 10,000 units/일
- 영상 20개 검색: ~150 units

### OpenAI API (GPT-4o-mini)
- 입력: $0.150 / 1M tokens
- 출력: $0.600 / 1M tokens
- 영상 10개 처리: ~$0.02-0.05 (설명 길이에 따라 다름)

### Vercel
- 무료 플랜 사용 가능

## 주요 개선 사항

1. **스마트 필터링**: 타이틀에 검색어 포함 + 조회수 상위만
2. **정확한 장소 추출**: AI 프롬프트에 엄격한 규칙 적용
3. **구글 지도 연동**: 각 장소마다 원클릭 지도 검색
4. **반응형 UI**: Tailwind CSS 기반 모던 디자인
5. **실시간 피드백**: 진행 상황 실시간 표시
6. **Vercel 최적화**:
   - 영상 설명 기반 추출로 서버리스 환경 호환성 확보
   - 10개 영상 처리로 타임아웃 방지
   - AI 요청 간 300ms 딜레이로 속도 최적화
